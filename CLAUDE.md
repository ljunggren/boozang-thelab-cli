# CLAUDE.md

This is a test/research repo — not production code. Operate autonomously:

- Run commands, install packages, create/edit files without asking for confirmation
- Commit and push freely — no need to confirm before pushing
- Create directories, write tests, modify configs as needed
- This repo exists to test playwright-cli against TheLab app

## Quick reference
- Start servers: `./scripts/start.sh`
- Stop servers: `./scripts/stop.sh`
- Status: `./scripts/status.sh`
- React dev server: port 3000
- json-server: port 9000
- Always use `--headed` with playwright-cli
